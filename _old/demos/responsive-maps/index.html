--- title: "Responsive Google Map Demo" summary: "Making a map responsive has its challenges. This FitVids.JS solves this issue with its customSelector property." ---

Learn how to create your own responsive map here.

Resize your browser and watch as the map on top keeps its shape and the map on bottom resizes with the browser.

Take a basic 425 x 350 pixel iframe



Apply fitVids.js to the iframe


$(document).ready(function() {
	$('.mapWrap').fitVids({ customSelector: "iframe[src*='maps.google.com']" });
});
